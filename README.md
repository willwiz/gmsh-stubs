# gmsh-stubs

Basic type stubs for [gmsh](https://gitlab.onelab.info/gmsh/gmsh)'s Python API based on docstrings.

## Features

- Stub only package
- Passes Pyright on strict
- Minor Ruff rules ignored

## Examples

- Minor edits of the gmsh examples for python name conventions

## Installation
```bash
pip install ./gmsh-stubs
```
